# 🎮 Сетевые Крестики-Нолики в Docker

Веб-приложение для игры в «Крестики-Нолики» по сети с интерактивным игровым лобби, анимацией игрального кубика и кастомизацией интерфейса. Проект разработан в рамках вузовской практики по контейнеризации приложений.

---

## ✨ Ключевые особенности проекта

*   **Архитектура Client-Server**: Чёткое разделение на независимый бэкенд (Python) и фронтенд (HTML/JS).
*   **Мгновенный отклик (WebSockets)**: Игроки видят ходы друг друга в реальном времени без перезагрузки страниц.
*   **Интерактивное Лобби**: При входе игроки вводят свои имена и выбирают уникальные аватарки-символы (Лиса 🦊, Ракета 🚀 и др.). Занятые аватарки автоматически блокируются для других участников.
*   **Честный розыгрыш (Dice Roll)**: Первый ход определяется броском игрального кубика на стороне сервера, что исключает читерство.
*   **Режим зрителя (Viewer Mode)**: Первые два подключившихся пользователя становятся игроками, остальные автоматически переключаются в режим просмотра матча.
*   **Кастомизация интерфейса**: Встроенный переключатель тем оформления (Классика ⚪, Космос 🚀, Лето 🌿).
*   **Полная контейнеризация**: Проект упакован в независимые Docker-контейнеры и разворачивается одной командой.

---

## 🛠️ Технологический стек

*   **Backend**: Python 3.10, FastAPI, Uvicorn, WebSockets.
*   **Frontend**: HTML5, CSS3 (Градиенты, Flexbox, Grid), JavaScript (Vanilla ES6).
*   **DevOps & Инструменты**: Docker, Docker Compose, Nginx (Alpine), Git, `uv` (пакетный менеджер).

## 🚀 Инструкция по запуску проекта

Для запуска игры на любом компьютере необходим установленный **Docker Desktop**.

### 1. Клонирование репозитория
Откройте терминал/командную строку и скачайте проект:
```bash
git clone https://github.com/laminaevelina/docker-tic-tac-toe.git
cd docker-tic-tac-toe
```

### 2. Запуск через Docker Compose
Выполните команду сборки и запуска в корневой папке проекта:
```bash
docker-compose up --build
```
Докер автоматически скачает необходимые образы, установит сетевые библиотеки Питона, настроит веб-сервер Nginx и свяжет контейнеры в единую сеть.

### 3. Открытие игры
После завершения сборки откройте браузер и перейдите по адресу:
👉 **http://127.0.0.1:8080**

*Чтобы сыграть матч самим с собой или протестировать сеть, просто откройте этот же адрес во второй вкладке браузера!*
